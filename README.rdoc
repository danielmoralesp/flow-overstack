== FLOWOVERSTACK (Clone StackOverFlow)

This app is stackoverlow clone with the flowing  features:

* User CRUD

If the user is athenticate can:

* Do Questions (CRUD)
* Answer each Question.
* Do Comments for questions and answers.
* Each user can vote each answer and each question one time.
* anyone can search questions and answers.


Heroku URL Deploy: https://flowoverstack-jch.herokuapp.com/
